# Number Guessing Game 🎮
A terminal-based Number Guessing Game built with Python.
The program randomly selects a number between 1 and 100
and challenges the player to guess it within a limited
number of attempts based on the chosen difficulty.

# Features
- 3 difficulty levels (Easy, Medium, Hard)
- Input validation for both difficulty and number guess
- Range validation (only accepts numbers between 1-100)
- Hint system — tells you if you're very close (within 5)
- Play again option without restarting the program
- Score tracker showing wins, losses and win rate

# How to Run
python main.py

# Tech Used
- Python 3
- Random module
